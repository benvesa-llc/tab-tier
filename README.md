# Tab Tier 🗂️

Smart tab lifecycle management for your browser.

Tab Tier automatically manages your browser tabs through a tiered lifecycle system (T0–T4), keeping your browser fast and organized without losing any tabs.

## Features

- **Tiered Tab Lifecycle (T0–T4):** Tabs are automatically categorized based on usage frequency and recency
- **Smart Tab Management:** Unused tabs are progressively moved to lower tiers
- **Tab Restoration:** Easily restore tabs from any tier when you need them
- **Memory Optimization:** Reduce browser memory usage by managing inactive tabs
- **Lightweight:** Minimal resource footprint, runs silently in the background

## How It Works

Tab Tier uses a tiered system to manage your tabs:

- **T0 (Active):** Currently active and recently used tabs
- **T1 (Warm):** Tabs used in the last few hours
- **T2 (Cool):** Tabs not used for a while
- **T3 (Cold):** Tabs inactive for a long time
- **T4 (Frozen):** Tabs stored for later, minimal resource usage

Tabs automatically move between tiers based on your usage patterns.

## Development
